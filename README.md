# service_queue_provider
using RabbitMQ, mailGun, CodeX , S3 storage

## Getting started

* install prisma 
    ```npm i prisma```
    ```npm i @prisma/client```
* run the command
    ```npx prisma migrate dev```
* then start the project
    ```node server.js```

* you can also run 
    ```npx prisma studio```
    to see database entries.

you can access our routes in swagger on ```/docs```
